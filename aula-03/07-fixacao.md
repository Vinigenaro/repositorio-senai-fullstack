# Perguntas Rápidas

1. Qual é a diferença entre condição e decisão?
   A condição é o que deve ser verificado para garantir que a decisão seja adequada.

2. Por que a ordem dos else if pode mudar o resultado?
   Pois certas condições devem ser verificadas antes de outras, por exemplo na entrada em um evento podemos conferir a idade mínima necessária e senão for real, se a pessoa pode entrar na área VIP, se eu conferir a idade e informar que "entrada aprovada!", a conferência de entrada VIP nunca será realizada.

3. Quando é melhor usar switch em vez de else if?
   Quando a variável possui respostas pré-definidas como tipos de produtos.

4. O que acontece se esquecer o break dentro do switch?
   Todos os outros casos abaixo serão executados.
