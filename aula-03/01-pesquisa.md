1. **O que é uma estrutura condicional em programação?**
Estrutura condicional é um recurso que permite executar ações diferentes de acordo com uma condição.

2. **Qual é a diferença entre usar apenas if e usar if/else?**
O `if` executa uma ação apenas quando a condição é verdadeira, enquanto o `if/else` define ações para os dois resultados possíveis.

3. **Quando usamos else if?**
Usamos `else if` quando precisamos testar várias condições diferentes em sequência.

4. **O que significa uma condição retornar true ou false?**
Significa que a condição foi avaliada como verdadeira (`true`) ou falsa (`false`).

5. **Qual é a diferença entre == e === em JavaScript? Qual deles é mais recomendado?**
`==` compara valores com conversão de tipos e `===` compara valores e tipos. O mais recomendado é o `===`.

6. **Para que servem os operadores &&, || e !?**
`&&` exige que todas as condições sejam verdadeiras, `||` exige que pelo menos uma seja verdadeira e `!` inverte o resultado lógico.

7. **O que o operador % faz? Por que ele é útil para descobrir se um número é par ou ímpar?**
O operador `%` retorna o resto da divisão. Ele é útil porque números pares têm resto `0` ao serem divididos por `2`.

8. **O que é switch case? Em que situação ele fica melhor do que vários else if?**
`switch case` é uma estrutura de seleção usada para comparar uma variável com vários valores. Ele é mais indicado quando há muitas opções fixas.

9. **Para que servem break e default dentro de um switch?**
`break` encerra a execução do caso atual e `default` define a ação executada quando nenhum caso é atendido.

10. **Por que é importante testar o mesmo programa com vários valores diferentes?**
É importante para verificar se o programa funciona corretamente em diferentes situações e entradas.
