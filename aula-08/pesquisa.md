## 1. O que é um objeto em programação?

É uma instância de uma classe. Ele armazena dados (propriedades) e possui comportamentos (métodos).

## 2. O que é uma propriedade de um objeto? Cite um exemplo.

É uma variável que guarda uma informação do objeto. Exemplo: em um objeto `Carro`, a propriedade `cor` pode ter o valor `preto`.

## 3. O que é um método? Qual a diferença entre método e propriedade?

Um método é uma função que pertence ao objeto e executa uma ação. A diferença é que o método realiza uma ação, enquanto a propriedade apenas armazena um valor.

## 4. O que é uma classe?

É um modelo que define quais propriedades e métodos os objetos terão.

## 5. Qual a diferença entre classe e objeto?

A classe é o modelo ou estrutura. O objeto é uma instância criada a partir dessa classe.

## 6. Para que serve o `constructor` dentro de uma classe?

Serve para inicializar o objeto, atribuindo valores às propriedades quando ele é criado.

## 7. O que significa a palavra `this` dentro de uma classe?

`this` faz referência ao próprio objeto, permitindo acessar suas propriedades e métodos.

## 8. O que a palavra `new` faz ao criar um objeto?

Ela cria uma nova instância da classe, reserva memória para o objeto e executa o `constructor`.

## 9. Em TypeScript, por que usamos tipos como `string`, `number` e `boolean`?

Para definir o tipo dos dados, evitar erros e tornar o código mais seguro e fácil de manter.
