1. **O que são operadores em programação?**
Operadores são símbolos que realizam operações sobre valores e variáveis.

2. **Para que servem os operadores aritméticos? Cite exemplos.**
Operadores aritméticos servem para fazer cálculos, como `+`, `-`, `*` e `/`.

3. **Qual a diferença entre `/` e `%` em JavaScript?**
`/` faz a divisão e `%` retorna o resto da divisão.

4. **Como descobrir se um número é par usando o operador `%`?**
Um número é par quando `numero % 2 === 0`.

5. **O que é uma expressão em programação?**
Uma expressão é uma combinação de valores, variáveis e operadores que produz um resultado.

6. **O que significa precedência de operadores?**
Precedência de operadores define a ordem em que as operações são executadas.

7. **Por que os parênteses podem mudar o resultado de uma expressão?**
Os parênteses alteram a ordem de execução das operações em uma expressão.

8. **Qual a diferença entre `=`, `==` e `===`?**
`=` atribui valor, `==` compara valores e `===` compara valores e tipos.

9. **Por que é recomendado usar `===` em JavaScript?**
É recomendado usar `===` porque ele evita comparações incorretas entre tipos diferentes.

10. **O que os operadores relacionais retornam: texto, número ou boolean?**
Operadores relacionais retornam valores booleanos (`true` ou `false`).

11. **Quando usamos o operador `>`? E o operador `>=`?**
`>` verifica se um valor é maior que outro e `>=` verifica se é maior ou igual.

12. **Quando usamos o operador `<`? E o operador `<=`?**
`<` verifica se um valor é menor que outro e `<=` verifica se é menor ou igual.

13. **Para que serve o operador lógico `&&`?**
`&&` retorna verdadeiro apenas quando todas as condições são verdadeiras.

14. **Para que serve o operador lógico `||`?**
`||` retorna verdadeiro quando pelo menos uma condição é verdadeira.

15. **Para que serve o operador lógico `!`?**
`!` inverte o valor lógico de uma condição.

16. **Qual a diferença entre uma condição simples e uma condição composta?**
Uma condição simples possui uma única comparação, enquanto uma composta possui duas ou mais.

17. **Explique uma situação real em que duas condições precisam ser verdadeiras ao mesmo tempo.**
Para entrar em uma prova, o aluno precisa estar inscrito e apresentar documento.

18. **Explique uma situação real em que apenas uma entre duas condições precisa ser verdadeira.**
Para ganhar desconto, o cliente pode ser estudante ou idoso.

19. **Como o `console.log()` pode ajudar a encontrar erros no código?**
`console.log()` ajuda a visualizar valores e identificar onde ocorre um erro.

20. **Por que devemos testar o mesmo programa com valores diferentes?**
Devemos testar com valores diferentes para verificar se o programa funciona em várias situações.
